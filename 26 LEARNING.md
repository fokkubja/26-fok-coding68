ขั้นตอนของโปรแกรมจัดการเวลา
Input
-กรอกจำนวนเวลาที่ต้องการเช่น เวลาในการทำงาน และเวลาในการพักผ่อน และอื่นๆ
Process
-จัดการแบ่งตารางเวลาเช่น ต้องการเวลาในการพักผ่อน2ชั่วโมง และ เวลาในการทำงาน4ชั่วโมง จะแบ่งเวลาออกเป็น 1ชั่วโมง30นาทีทำงาน 1 ชั่วโมงในการพักผ่อน 1ชั่วโมง30นาทีทำงาน 1 ชั่วโมงในการพักผ่อน
Output
-ตารางเวลาที่จัดการให้ตามเวลาที่กรอกไว้

1.ไม่ยากเกินไปสำหรับโฟล์คครับ

3.ยากที่สุดคือขั้นตอนในการ Process แก้ไขได้โดยการใช้สูตรคณิตศาสตร์

โค้ด โปรแกรม tdee
w = int(input("น้ำหนัก(กก.)"))
cm = int(input("ส่วนสูง(ซม.)"))
a = int(input("อายุ"))
s = int(input("เพศ(เช่น 4 ผู็ชาย 5 ผู็หญิง)"))
wo =int(input("หากไม่ต้องการออกกำลังกาย กรอก1 ออกกำลังกายเบาๆ กรอก2 ออกกำลังกายอย่างจริงจัง กรอก3"))

bmr = 0 
if s == 4:
    bmr = (13.7 * w) + (5 * cm) - (6.8 * a) + 66
elif s == 5:
    bmr = (9.6 * w) + (1.8 * cm) - (4.7 * a) + 665
else:
    print("การป้อนเพศไม่ถูกต้อง (ต้องเป็น 4 หรือ 5) BMR จะไม่ถูกคำนวณ")

tdee = 0
if bmr > 0: 
    if wo == 1:
        tdee = bmr * 1.2
        print("ค่าTDEEคือ", tdee)
    elif wo == 2:
        tdee = bmr * 1.375
        print("ค่าTDEEคือ", tdee)
    elif wo == 3:
        tdee = bmr * 1.55
        print("ค่าTDEEคือ", tdee)
else:
    print("ไม่สามารถคำนวณ TDEE ได้เนื่องจาก BMR ไม่ถูกต้อง")
